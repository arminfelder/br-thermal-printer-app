# Media Info

Contains extracted media definitions as described in the TD2000 raster manual.

## Reference

- **Document:** [Raster Command Reference](https://download.brother.com/welcome/docp000750/cv_td2000_eng_raster_101.pdf)
- **Version:** 1.01
- **Page:** 28

## Description

This directory contains media type definitions used for thermal printer configuration,
including paper sizes, print densities, and other media-related parameters.
