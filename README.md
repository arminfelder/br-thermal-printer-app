# br-thermal
Unofficial Driver for Brother Thermal Printers

> Status: This driver is a work in progress and not yet feature-complete. Expect missing features and potential issues.

## build
- mkdir build
- cd build
- cmake ..
- make

## Run
- ./server

## currently supported devices
- TD-2020/2120N/2125N/2125NWB
- TD-2030A/2130N/2135N/2135NWB
